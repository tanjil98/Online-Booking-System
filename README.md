A fully responsive and dynamic bus ticket booking platform designed to provide a seamless user experience. This project allows users to select seats, calculate costs, apply discount coupons, and book tickets instantly.

üåü Features
-Interactive Seat Selection: Users can select up to 4 seats from a visual bus layout.

-Dynamic Cost Calculation: Real-time updates for seat count, individual ticket price, and total price.

-Coupon System: - Apply coupon code NEW15 to get a 15% discount.

-Apply coupon code Couple 20 to get a 20% discount (requires purchase of 4 seats).

-Validation: - Prevents selecting more than 4 seats.

-"Next" button remains disabled until a seat is selected and a phone number is provided.

-Success Modal: A confirmation modal appears upon successful booking.

-Responsive Design: Fully optimized for mobile, tablet, and desktop views using Tailwind CSS.
